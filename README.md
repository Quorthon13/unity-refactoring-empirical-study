# Unity Refactoring Empirical Study

This repository contains the notebooks and datasets produced for an empirical study on refactoring practices in C# projects developed with the Unity engine.

The materials provided here support the analyses and results presented in the associated bachelor's thesis.

## Contents

- Analysis notebooks
- Processed datasets (CSV)
- Research artifacts generated during the study

## Reproducibility

The notebooks can be executed to reproduce the analyses described in the thesis. A Python environment with standard data analysis libraries (e.g., pandas, numpy, matplotlib) is recommended.

## Citation

Rodrigues, Diego Demétrio Santos. "Refatoração em Realidade Virtual: Um Estudo Empírico em Projetos Unity." Bachelor's Thesis, Universidade Federal de Ouro Preto, 2026.
